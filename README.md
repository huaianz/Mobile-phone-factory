# Mobile-phone-factory

游戏流程：

1.首先是点击经济预测，进行政府措施，先提交再开始政府措施，这个时候会生成相应数量的订单

2.点击订单竞标，点击开始竞标后，在模拟工具（手机）里选择要竞标的订单ID和相应报价，一键修改是防止输入的内容有误，点击提交后即可，开始竞标后会有计时器计时，设置是十秒十秒的重复，这样每个人轮流竞拍（但是好像没写好），结束竞拍后会在Console里显示中标的订单，点击延迟竞拍，可继续竞拍，点击重新竞拍会刷新订单

3.点击贷款，需要先点击是否参与，接下来开始竞标输入贷款金额，再提交即可

4.芯片采购和零部件采购，首先需要根据我们中标的订单手机ID去组装生产页面的手机生产表查看手机名字，接着根据相应机型去芯片采购和零部件采购显示信息中的适配机型购买相应零件，有的零件适合多个手机，具体可以看Project中Resources中的phone.json文件，有相应ID

5.土地竞标，同样先点击参与，输入土地每年的租金价格，
